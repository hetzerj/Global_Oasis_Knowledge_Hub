## How to Run Locally  
1. Install **R** and **RStudio** (if not already installed).  
2. Install the **shiny** package if you haven't:  
   ```r
   install.packages("shiny")
   ```
3. Clone or download this repository.  
4. Open **`app.R`** in RStudio and run:  
   ```r
   shiny::runApp()
   ```
